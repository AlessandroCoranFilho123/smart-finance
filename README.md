# ProjetoFinancas

Projeto pessoal para desenvolver minhas habilidades em Java, e também me ajudar a economizar dinheiro

## Funcionalidades

- Cadastro de transações
- Criação de metas

## Tecnologias

- Java 21.0.9 (LTS)
- JavaFX 21.0.10
- Css
- Maven
- jlink / jpackage
